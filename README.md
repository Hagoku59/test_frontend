## Тестовое задание

Тестовое задание по Frontend - таблица с данными о персонажах.

## Использование

### Установка зависимостей
Для установки зависимостей, выполните команду:
```sh
$ npm install
```
### Запуск клиентского приложения
Чтобы запустить приложение, выполните команду:
```sh
npm start
```
